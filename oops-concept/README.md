# JavaScript OOP Examples

This project demonstrates key concepts of Object-Oriented Programming (OOP) in JavaScript, specifically focusing on encapsulation and inheritance.

## Files Overview

- **encapsulation.js**: This file contains a `Person` class that demonstrates encapsulation by using private properties for `name` and `age`. Access to these properties is controlled through public methods.

- **inheritance.js**: This file features an `Employee` class that extends the `Person` class. It showcases how inheritance allows for the addition of new properties and methods, specifically a `jobTitle` property.

- **oops-class.js**: This is the entry point of the examples. It imports the `Person` and `Employee` classes and creates instances to illustrate how encapsulation and inheritance work together.

## How to Run the Examples

1. Clone the repository or download the project files.
2. Navigate to the project directory in your terminal.
3. Run the examples using Node.js by executing the following command:

   ```
   node oops-class.js
   ```

This will demonstrate the functionality of encapsulation and inheritance in JavaScript.
